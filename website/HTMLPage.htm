Key Mapper Help

User scenarios
----------------------

1) Mike is fed up of accidentally pressing Num Lock or Insert - keys
which he never uses otherwise, but which mess up his typing. He needs
to have Num Lock set On, though.

2) Sue hates accidentally pressing Caps Lock when she wants Shift. She
hardly ever uses Caps Lock.

3) Mark in annoyed that his work keyboard doesn't have hardware volume
up/down buttons like his Mac keyboard at home.

4) Mary is always looking for her email program and browser in the
taskbar. She wishes she could just press a button to make them appear
whether they are open or not.

5) Sandy is frustrated that when using Boot Camp or virtualization to
run Windows on her Mac that there's no Print Screen or Scroll Lock
keys (she uses Excel all the time, and it happens to be the only
program on Earth that actually takes notice of Scroll Lock). She also
gets fed up with pressing Command - expecting it to behave like
Control - when in fact it's the Windows key and opens the Start Menu.

6) Andy keeps pressing "Power Off" on his Extended Keyboard because
it's right next to the Delete key, causing much annoyance and
inconvenience.

7) Kevin has broken the 6 key on his laptop. He'd like to remap
another key to it, like F6, rather than have to buy and fit a new
keyboard.

8) George hates Sarah and wants to mess with her head, so he wants to
remap half her keys and disables the other half.

Using Key Mapper lets keys be disabled or remapped to other keys.
Mike can disable Num Lock and Insert completely.
Sue can remap Caps Lock to Left Shift and Scroll Lock to Caps Lock.
Mark can make F11 and F12 Volume Down and Up
Mary can remap F11 to 'Email' and F12 to 'Browser'
Sandy can remap one of her function keys to Print Screen and the
Command key to Control, and so on.
George gets fired..

Installing and use
-----------------------

1) Download and run the install program from
http://justkeepswimming.net/keymapper/install.html - this should work
in Firefox or IE or Safari. You'll get some warnings about "Unknown
Publisher" but you know it's me really.
2) Run it - it will be under Rose Hill Solutions/Key Mapper in your Start Menu
3) Drag keys off the virtual keyboard to disable them, or drop them
onto another key to remap it
4) If you don't want to actually create any mappings, clear them
before exiting the program!
5) If you have mapped or disabled anything, you have to log off and on
again so it takes effect.

If you install via this route, you'll receive updates automatically
and the install program will check to see you have the program's
prerequisites installed. However, each time you run it, you'll get a
slightly annoying window first - if that bothers you, use the manual
install below: if you like, you can use the install page to check the
prerequisites and then do a manual install from below.

That's all you really need to know to use the program. Feel free to
read on though.

More on using the program
--------------------------------------

To disable a key, drag it off the keyboard.
Example: Drag Num Lock off the keyboard to disable it

To change what a key does, drop the key you want to replace it with onto it.
Example: Drag Left Shift onto Caps Lock. Caps Lock will be mapped to Left Shift

To re-enable - or un-map - a mapped or disabled key, drag it off the keyboard.

To disable or remap a key that isn't on the virtual keyboard, create a
mapping from the Mappings menu.
You can either choose the key by pressing it, or from the key lists.

Example: Choose the Power Off key from the lists, then choose Disable.

You can also double-click the key you want to change or disable:
Example: Double-click on F11, and choose Volume Down from the Media Keys group.

To change a key to one that isn't in the lists, double-click the key
then use Capture to set the target.
Example: To map F5 to the "Have A Cow" button on your Simpsons
Extended keyboard, double click F5, click the Capture button to start
capturing, then press the "Have  A Cow" button.

To delete a mapping that isn't visible on the keyboard:
Show the Mapping List form from Key Mapper's Windows menu, and click
Delete next to the mapping you want to remove.

To change the key colours: Show the Colour Map from Key Mapper's
Windows menu, then double-click on the key you want to change. Enter
new values in the Colour Matrix presented or click Random! Right-click
on the colour map form to reset the colours or show all the buttons.

Do the changes take effect straight away?
--------------------------------------------------

No. To complete the mapping - or unmapping - you will need to log off
and log back on again. Using CTRL-ALT-DELETE is the easiest way to log
off, once you've closed things down and saved any documents.
Restarting will work just as well, and if you're using Windows 2000
(or creating Boot mappings which apply to all users) then restarting
is required instead of logging off.

Why is Key Mapper different from all the other programs that do this?
----------------------------------------------------------------------------------------

They other programs write 'boot mappings' (which apply to all users of
the computer and require a restart to activate) whereas Key Mapper
allows you to make mappings on a per-user basis - functionality
Microsoft introduced in Windows XP but forgot to tell anyone about or
document in any way.

Issues
-----------
Some keyboards have extra keys - e.g. Power, Email, Web Browser - and
the keyboard driver intercepts these keys before they reach Windows,
which means Key Mapper can't capture them and Windows can't remap
them. If you have a keyboard with these keys, can you try to see if a)
they can be captured and b) they can be remapped or disabled?

If you use "Fast User Switching" to switch between users, then you'll
need to create boot mappings instead of user mappings - as when you
use Fast User Switching to switch to an existing user then the
per-user key mappings are lost. This is (in my opinion) a bug in
Windows, but they would probably say that they never documented the
per-user mappings anyway so meh.

Installing the program manually
-------------------------------------------

The RC1 build can be found at
http://code.google.com/p/keymapper/downloads/list along with the
source code used to build it. If you want to build the program
yourself, use Microsoft Visual Studio 2008 Express Edition which is a
free download.

The program requires the Microsoft .Net Framework to be installed. It
requires version 2.0, but will happily use 3.0 or 3.5. However, it's
likely that most of you won't have the framework installed (Microsoft
don't include it by default) and you'll need to download it. The later
versions are bigger: 2.0 is available from
http://www.microsoft.com/downloads/details.aspx?FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5&displaylang=en
and is 22.4 MB. The .NET framework itself requires Windows Installer
3.1, if your PC gets automatic updates from Microsoft you'll already
have this though.

If you have Vista, the framework is installed by default.

If you run the program without the framework installed you'll get a
"Mscoree.dll Could Not Be Found" error.

Why does the program warn me off trying to remap Control, Alt or Delete?
-------------------------------------------------------------------------------------------------------------

Because that's the key sequence you need to log onto Windows. You can
still do it if you really want to.

What other keys are there I can use for mappings?
------------------------------------------------------------

There are a whole bunch of keys that can be used in mappings. Some
simply don't work (Pause/Break can't be remapped, the extended power
functions can't be used without specific keyboard drivers) and many
are of limited use, but there are a few which are interesting and
which work.

Browser and Email  Keys
-------------------------------

As I mentioned above, keys can be mapped for Browser and Email. They
both invoke your default program, activating a running existence or
starting a new one as required. This works on all the programs I've
tried - Internet Explorer, Firefox, and Opera for browsers, Outlook
Express Outlook and Thunderbird for email.

There are also Next and Back and Refresh buttons. If you use Internet
Explorer, these work whether it is the current application or not, but
other browsers will only react to these when they are currently
active.

Media Keys
-----------------

Volume Down, Up and Mute work whatever program you're using as they
are telling Windows to be louder or quieter, not a given application.

Next Track and Previous Track should work in most media programs: if
you use Windows Media Player, it will switch tracks whether it's the
foremost program or in the background, but iTunes only recognizes
these when it's the active program.

What else does the program do?
--------------------------------------

You can toggle the status of Caps, Num and Scroll lock from the menu.

You can explore all the international keyboards installed on your PC
(quite good fun if you use Vista as lots of exotic locales are
installed by default)
